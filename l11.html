Website

Er werd gevraagd om nog een paar zaken aan onze website toe te voegen. Voor mij waren dat de volgende punten:

  • Op elke pagina moet mijn naam en groep vermeld staan.
  • Op elke pagina moet een footer geplaatst worden met een link naar de laatst gepubliceerde versie van mijn code op GitHub.
  • Op de homepage moet een link staan om de gecomprimeerde pdf van de ontwerpdocumentatie te downloaden.

Deze punten heb ik in orde gebracht. Daarnaast heb ik Tailwind correct ingesteld. Voorheen werd Tailwind rechtstreeks aangeroepen in mijn HTML (de website), maar nu wordt het via Node aangeroepen.
