Hoe ben ik begonnen?

Tijdens de les heb ik mijn bestaande GitHub-account gekoppeld aan Netlify. Daarna ben ik meteen gestart met het ontwerpen van een eenvoudige eerste versie van mijn website. Omdat ik al wat ervaring heb met HTML en CSS, verliep dit vrij vlot.

Vervolgens heb ik de bestanden in een GitHub-repository geplaatst en de site via Netlify online gezet.

HTML-pagina

De eerste opmaak van mijn website is voorlopig eenvoudig gehouden.
Ik heb:

  • Een ander lettertype ingesteld.
  • Kleuren aangepast.
  • een achtergrondpatroon toegevoegd
  • een flexbox-layout gemaakt waarin alle lesweken staan
  • per lesweek een “card” ontworpen die doorlinkt naar een aparte pagina

Het is een basisstructuur, maar wel een die al goed werkt als startpunt.

Eindstand

Het verslag is dit keer redelijk kort, omdat het niet heel veel werk was om alles in orde te krijgen. Ik heb wel al wat CSS toegevoegd, maar ik vermoed dat dit misschien nog niet volledig de bedoeling was op dit moment in het project.
